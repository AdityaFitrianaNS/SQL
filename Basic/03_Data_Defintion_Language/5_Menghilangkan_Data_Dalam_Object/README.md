## **Menghapus semua Record atau Data didalam sebuah tabel**

  `TRUNCATE`

  Bentuk Umum : `TRUNCATE TABLE ` + Nama Objek + `;`

  Note 
  > Jika terdapat lebih dari 1 kata maka gunakan tanda ``_`` sebagai pengganti spasi. 
  
  Contoh : 
  - `TRUNCATE TABLE Mahasiswa;`
